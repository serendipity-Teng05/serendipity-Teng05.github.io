# 邓才增 · 个人主页

基于 HTML/CSS/JS 的简约科技风个人主页，用于研究生复试简历展示。

## 技术栈

- HTML5
- CSS3（CSS 变量、Grid、Flexbox、渐变、动画）
- JavaScript（QRCode.js 二维码生成）
- 响应式设计（移动端适配）

## 页面结构

```
├── 导航栏          # 锚点跳转：关于我 / 项目 / 竞赛 / 联系
├── Hero            # 姓名、专业、学校
├── 自我介绍         # 个人简介与学术背景
├── 教育背景         # GPA、排名、核心课程
├── 初试成绩         # 五科成绩卡片展示
├── 技术栈           # 标签云，核心技能高亮
├── 科研与项目经历    # 3 个项目的时间线展示
├── 竞赛获奖         # 省级/校级奖项网格
├── 校园经历         # 学生工作经历
└── 联系方式         # 二维码 + 邮箱/电话/学校信息
```

## 设计风格

- **深色主题**：底色 `#0a0e17`，配合青蓝/紫色科技感高亮
- **氛围光效**：网格背景 + 顶部/底部径向渐变光斑
- **卡片布局**：半透明边框卡片，hover 微亮交互
- **全部响应式**：手机端自适应网格切换

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构与内容 |
| `style.css` | 全部样式（CSS 变量定义主题色） |
| `研究生复试邓才增.pdf` | 原始简历文档 |

## 使用方法

直接在浏览器打开 `index.html`，或使用任意静态服务器：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

## 自定义

编辑 `style.css` 中的 CSS 变量可快速调整整体配色：

```css
:root {
  --accent: #06b6d4;   /* 主色调 */
  --accent2: #8b5cf6;  /* 辅助色 */
  --bg: #0a0e17;       /* 背景色 */
}
```
